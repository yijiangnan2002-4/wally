Detailed Description

This structure defines additional data format of the notify session nonce message.

This structure defines the format of SASS MAC verify data.

This structure defines one item of address to session nonce, account key id and bitmap index.

This structure defines the data format of RHO old agent send to new agent.

This structure defines the data format of the notify connection status message.

This structure defines the data format of the notify multipoint-switch event.

This structure defines the data format of the notify switching preference.

This structure defines the received data format of setting switching preference from seeker.

This structure defines the received data format of setting multipoint state from seeker.

This structure defines the received data format of notify capability.

typedef void(* bsp_gsensor_key_callback_t) (uint8_t key_data, bsp_gsensor_key_event_t event, void *user_data)

This defines the callback function prototype.

User should register a callback function when using Gsensor key, this function will be called in external interrupt service routine after a Gsensor key is tapped.

Parameters
[in]key_datais used to identify a specific key.
[in]eventis the key event for the current key status, user can get the key status from this parameter, please refer to bsp_gsensor_key_event_t for details about the event type.
[in]user_datais a user-defined parameter provided by bsp_gsenosr_key_init().

Typedef Documentation

typedef void(* mux_callback_t) (mux_handle_t handle, mux_event_t event, uint32_t data_info, void *user_data)

MUX callback function type When event == MUX_EVENT_READY_TO_READ or MUX_EVENT_TRANSPARENT_READ, data_info means: there have one package data, data_info is receive data len.


When event != MUX_EVENT_READY_TO_READ or MUX_EVENT_TRANSPARENT_READ, data_info means: there have a data info from isr.
user_data means: The user data brought in by the user when doing mux_open.

typedef uint32_t mux_handle_t

MUX handle type.


bit[16:31]:magic number.
bit[8:15]:user_id.
bit[0:7]:mux port.

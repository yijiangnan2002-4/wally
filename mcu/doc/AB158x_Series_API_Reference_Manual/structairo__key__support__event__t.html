Detailed Description

The event support configuration structure of the airokey framework.

Field Documentation

uint8_t long_level

Supported long press level, the max level is 3, user can extend the max level.


0: No support.

uint8_t long_repeat

Supported long repeat.


1: Support repeat event after long press event.
0: No support.

uint8_t multiple_click

Supported multiple clicks, the max level is 3, user can extend the max level.


0: No support.

uint8_t press_release

1: Support hardware press and release event.


0: No support.

uint8_t slong

If support short long event, the long press level should be at least one.


Short long event indicates that a long press event occurs within a specified period of time after a short-click.
1: Support short long event.
0: No support.

uint8_t slong_repeat

1: Repeat event.


0: No support.

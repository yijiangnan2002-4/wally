#define BT_AWS_MCE_AGENT_STATE_ACTIVE   0XE0

Active Agent State.

The Agent enters this state after the Partner successfully active at this link>

#define BT_AWS_MCE_AGENT_STATE_ATTACHED   0x60

Attached Agent State.

The Agent enters this state after the Partner successfully connected with Agent at this link.

#define BT_AWS_MCE_AGENT_STATE_CONNECTABLE   0x50

Connectable Agent State.

In this state, the Agent is accessible by the Partner, and the Partner can scan for the Agent and connects with Agent.

#define BT_AWS_MCE_AGENT_STATE_INACTIVE   0x40

Inactive Agent State.

In this state, the Agent is unaccessible by the Partner.

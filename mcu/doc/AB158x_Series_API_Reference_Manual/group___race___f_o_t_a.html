Overview

Race FOTA is a sub-module of the race command module.

It provides the basic download flow of FOTA over SPP, BLE and so on.

RACE_ERRCODE race_fota_cancel ( void  )

Function for cancelling FOTA.

There are race_fota and Bisto_fota. This API can only cancel race_fota.

Returns
RACE_ERRCODE_SUCCESS succeed; otherwise, fail.
race_fota_mode_enum race_fota_get_fota_mode ( void  )

Function for getting the current FOTA mode.

Please be aware that the return value is valid only when race cmd FOTA is running.

Returns
the current FOTA mode.
bool race_fota_is_active_mode ( void  )

Function for checking if the current FOTA mode is the active mode or not.

There are the active mode and the background mode.

Returns
TRUE It is the active mode; FALSE It is the background mode. The return value is valid only when race fota is running.
bool race_fota_is_busy ( void  )

The function is used to check whether Race FOTA is busy or not.

Generally, Race FOTA is considered to be in busy state when there is data in the FOTA partition.

Returns
true means Race FOTA is busy, false means it is idle.

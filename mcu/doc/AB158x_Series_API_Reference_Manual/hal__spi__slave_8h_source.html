712 /********************* SPI Slave FSM *******************************
713 **cmd\status PWROFF PWRON ConfigRead(CR) ConfigWrite(CW)
714 ** Poweroff N Y Abnormal Abnormal
715 ** Poweron Y N N N
716 **ConfigRead N Y Abnormal Abnormal
717 ** Read N N Y Abnormal
718 **ConfigWrite N Y Abnormal Abnormal
719 ** Write N N Abnormal Y
720 
721 Note: N: invalid operation.
722  Y: the operation completed successfully.
723  Abnormal: an error occurred during the operation.
724 *******************************************************************/

Enumerator

The audio_transmitter state is idle, or is stopped.

AUDIO_TRANSMITTER_STATE_STARTING 

The audio_transmitter state is going to start, has not started yet.

AUDIO_TRANSMITTER_STATE_STARTED 

The audio_transmitter state is started, finish start.

AUDIO_TRANSMITTER_STATE_STOPING 

The audio_transmitter state is going to stop, is not stopped yet.

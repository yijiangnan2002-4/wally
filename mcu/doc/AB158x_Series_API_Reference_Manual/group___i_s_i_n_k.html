Overview

This section introduces the ISINK APIs including terms and acronyms, supported features, software architecture, how to use this module, ISINK function groups, enums, structures and functions.

Users can use this API to adjust different current output settings through ISINK registers. For example, to adjust the backlight brightness.

Terms and acronyms

Terms Details
ISINK Current Sink that provides different current outputs depending on the PMIC setting.
PMIC Power Management Integrated Chip. For an introduction to the Power Management Integrated Chip, please refer to Power management integrated circuit Wikipedia

How to use this module

-The ISINK breath mode configure parameter show as below:

-The ISINK pwm mode configure parameter show as below:

  • The user can set ISINK to operate in a different mode. The steps are shown below:
    • Step1: Call hal_isink_init() to initialize the ISINK module.
    • Step2: Call hal_isink_configure() to configure ISINK mode and setting.
    • Step3: Call hal_isink_start() to trigger ISINK HW start output signal.
    • Step4: Call hal_isink_stop() to trigger ISINK HW stop output signal.
    • Step5: Call hal_isink_deinit() to de-initialize ISINK module.
    • Breath mode sample code:
      hal_isink_config_t isink;
      isink.mode = HAL_ISINK_MODE_BREATH;
      isink.config.breath_mode.darker_to_lighter_time1 = HAL_ISINK_TIME_0_123_SECOND;
      isink.config.breath_mode.darker_to_lighter_time2 = HAL_ISINK_TIME_0_123_SECOND;
      isink.config.breath_mode.lightest_time = HAL_ISINK_TIME_0_123_SECOND;
      isink.config.breath_mode.lighter_to_darker_time1 = HAL_ISINK_TIME_0_123_SECOND;
      isink.config.breath_mode.lighter_to_darker_time2 = HAL_ISINK_TIME_0_123_SECOND;
      isink.config.breath_mode.darkest_time = HAL_ISINK_TIME_0_246_SECOND;
      if(HAL_ISINK_STATUS_OK != hal_isink_init(HAL_ISINK_CHANNEL_0)) { //Initialize the ISINK.
      //Error handler
      }
      if(HAL_ISINK_STATUS_OK != hal_isink_configure(HAL_ISINK_CHANNEL_0,&isink)) { //Configure ISINK mode and setting.
      //Error handler
      }
      if(HAL_ISINK_STATUS_OK != hal_isink_start(HAL_ISINK_CHANNEL_0)) { //Trigger ISINK execution.
      //Error handler
      }
      if(HAL_ISINK_STATUS_OK != hal_isink_stop(HAL_ISINK_CHANNEL_0)) { //Stop ISINK.
      //Error handler
      }
      if(HAL_ISINK_STATUS_OK != hal_isink_deinit(HAL_ISINK_CHANNEL_0)) { //Deinitialize the ISINK module.
      //Error handler
      }

Overview

Race event is a sub-module of the race cmd module.

It provides an event mechanism for other modules to receive and process the race events they need to.

Typedef Documentation

typedef RACE_ERRCODE(* race_event_callback) (int32_t register_id, race_event_type_enum event_type, void *param, void *user_data)

The callback prototype used in race_event_register().

When any race event occurs, the callback will be invoked.

Parameters
[in]register_idThe same register_id obtained when calling race_event_register().
[in]event_typeThe type of the event occurs
[in]paramThe parameter of the event
[in]user_dataThe same user_data input in race_event_register()
Returns
RACE_ERRCODE_SUCCESS succeed; otherwise, fail.

enum hal_i2s_lr_swap_t

Enable or disable data swapping between right and left channels of the I2S link.


This function only works when the sample width of the I2S is 16 bits.

Enumerator
HAL_I2S_LR_SWAP_DISABLE 

Disable the data swapping.

HAL_I2S_LR_SWAP_ENABLE 

Enable the data swapping.

enum airo_key_event_t

All the supported events of the airokey framework.

Enumerator
AIRO_KEY_RELEASE 

A key is hardware released.

AIRO_KEY_SHORT_CLICK 

A key is short-clicked.

AIRO_KEY_DOUBLE_CLICK 

A key is double-clicked.

AIRO_KEY_TRIPLE_CLICK 

A key is triple-clicked.

AIRO_KEY_LONG_PRESS_1 

A key is long pressed.

AIRO_KEY_LONG_PRESS_2 

A key is long pressed.

AIRO_KEY_LONG_PRESS_3 

A key is long pressed.

AIRO_KEY_SLONG 

A key is short-clicked and then long pressed.

AIRO_KEY_DLONG 

A key is double-clicked and then long pressed.

AIRO_KEY_REPEAT 

A key remains pressed after the last long press event is issued.

AIRO_KEY_PRESS 

A key is hardware pressed.

AIRO_KEY_LONG_PRESS_RELEASE_1 

A special event, a key is released after AIRO_KEY_LONG_PRESS_1.

This event will not be used in state machine.

AIRO_KEY_LONG_PRESS_RELEASE_2 

A special event, a key is released after AIRO_KEY_LONG_PRESS_2.

This event will not be used in state machine.

AIRO_KEY_LONG_PRESS_RELEASE_3 

A special event, a key is released after AIRO_KEY_LONG_PRESS_3.

This event will not be used in state machine.

AIRO_KEY_SLONG_RELEASE 

A special event, a key is released after AIRO_KEY_SLONG.

This event will not be used in state machine.

AIRO_KEY_DLONG_RELEASE 

A special event, a key is released after AIRO_KEY_DLONG.

This event will not be used in state machine.

AIRO_KEY_INVALID 

A key is pressed but it does not match any event.

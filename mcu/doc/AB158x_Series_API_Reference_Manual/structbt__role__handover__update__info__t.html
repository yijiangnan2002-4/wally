Detailed Description

This structure defines the parameters when RHO notify the users to update their context.

Field Documentation

const bt_bd_addr_t* addr

The peer address before RHO.

In Agent part, it is the phone's address. In Partner part, it is Agent's address.

const void* data

The RHO data buffer pointer for the specific user.

It is NULL in new Partner part. The buffer pointer is not guaranteed to be four-byte aligned.

bool is_active

The Partner attach state of this address.

uint8_t length

The RHO data length for the specific user.

It is 0 in new Partner part.

const bt_aws_mce_role_handover_profile_t* profile_info

The new profile information after RHO.

It is NULL in new Partner part.

bt_aws_mce_role_t role

The role before RHO.

In Agent part, it is Agent role. In Partner part, it is Partner role.

Overview

Race cmd module provides multiple ways to communicate with the smartphone or the PC by using commands with a specific format.

It supports the communication using UART, SPP, BLE and so on.

RACE_ERRCODE race_send_msg ( race_general_msg_t *  msg)

This function is used for the RACE module.

The message will be copied to the message queue and should be freed after this API returns if it is allocated dynamically.

Parameters
[in]msgIt is the data structure for RACE to send the message, including the message ID, device and message pointer.
Returns
RACE_ERRCODE_SUCCESS the RACE sent the message successfully.
Note
The race_send_msg() can select the UART, USB or BT SPP server/client port through port service for data transmission.

Overview
  • This is not an example project. This project is a bootloader for the chip. The developer can refer to this project to understand how to build a bootloader.
Hardware and software environment
  • Supported platform
    • Airoha IoT SDK for BT Audio Evaluation Board (EVB).
    • Airoha IoT SDK for BT Audio Evaluation Kit (EVK).
Directory contents
  • Source and header files
    • inc/hal_feature_config.h: The feature configuration file.
    • src/custom_fota.c: Fota update function customization.
    • GCC/startup_bootloader_"board".s: The startup file for GCC.
    • GCC/syscalls.c: The minimal implementation of the system calls.
  • Project configuration files using GCC
    • GCC/feature.mk: The feature configuration file.
    • GCC/Makefile: The Makefile.
    • GCC/tcm_"board".ld: The linker script.
Run the example
  • Connect board to the PC with serial port cable.
  • Build bootloader and an example project with the command, "./build.sh <board> example_project_name bl" under the SDK/mcu root folder and download the binary file to the EVB.
  • Power on the EVB and bootloader log are shown on serial tool.

Detailed Description

The enviroment mode parameters structure.

Field Documentation

U8 beamforming_switch

Beamforming function switch.

This control item is not used when “Mode Table Control Beamforming Switch” is off. 0(off), 1(on).

U8 lowcut_switch_l

Low-cut switch for left device to apply low-cut gain.

0(off), 1(on).

U8 lowcut_switch_r

Low-cut switch for right device to apply low-cut gain.

0(off), 1(on).

U8 mfa_switch_l

Reserved.

This parameter is not working.

U8 mfa_switch_r

Reserved.

This parameter is not working.

U8 nr_level

NR (Noise reduction) level with range [0, 31].

U8 nr_switch

NR (Noise reduction) function switch.

NR reduces stable background noise. 0(off), 1(on).

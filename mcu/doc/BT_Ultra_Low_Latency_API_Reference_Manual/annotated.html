Data Structures
Here are the data structures with brief descriptions:
 Cbt_ull_client_sidetone_switch_tThis structure defined the parameters of BT_ULL_ACTION_SET_CLIENT_SIDETONE_SWITCH which is used for user set client uplink sidetone on/off
 Cbt_ull_init_critical_user_data_tThis structure defines the parameters of BT_ULL_ACTION_INIT_CRITICAL_USER_DATA, which is used for user initiate critical data context
 Cbt_ull_latency_tThis structure defines the parameters of BT_ULL_ACTION_SET_STREAMING_LATENCY, which is used to set the streaming audio playback latency
 Cbt_ull_mix_ratio_tThis structure defines the parameters of BT_ULL_ACTION_SET_STREAMING_MIX_RATIO, which is used to set multi streaming mix ratio
 Cbt_ull_pairing_complete_ind_tThis structure defines the parameters of event BT_ULL_EVENT_PAIRING_COMPLETE_IND which indicates the result of ULL pairing
 Cbt_ull_pairing_info_tDefine the ULL pairing parameters
 Cbt_ull_ratio_tThis structure defines the specified streaming mix ratio
 Cbt_ull_rx_critical_user_data_tThis structure defines the parameters of BT_ULL_EVENT_RX_CRITICAL_USER_DATA_IND, which is used for user to receive the critical data
 Cbt_ull_sample_rate_tThis structure defines the parameters of action BT_ULL_ACTION_SET_STREAMING_SAMPLE_RATE which is used to set the sample rate for the streaming (speaker/microphone)
 Cbt_ull_streaming_info_tThis structure is the return of bt_ull_get_streaming_info which is used to get the specified streaming detail info
 Cbt_ull_streaming_sample_channel_tThis structure defines the parameters of usb streaming sample channel information
 Cbt_ull_streaming_sample_size_tThis structure defines the parameters of usb streaming sample size information
 Cbt_ull_streaming_tThis structure defines the streaming info
 Cbt_ull_tx_critical_user_data_tThis structure defines the parameters of BT_ULL_ACTION_TX_CRITICAL_USER_DATA, which is used for user to send flushable critical data between client and server
 Cbt_ull_user_data_tThis structure defines the parameters of BT_ULL_ACTION_TX_USER_DATA, which is used for transmission user raw data between client with server
 Cbt_ull_volume_tThis structure defines the parameters of action BT_ULL_ACTION_SET_STREAMING_VOLUME which is used to control the streaming (speaker/microphone) volume

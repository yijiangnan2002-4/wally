Smart charger case module usage guide

Brief:          This module is the smart charger case interface. 
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/smart_charger/module.mk" in your GCC project Makefile.

Dependency:     To enable it, please define AIR_SMART_CHARGER_ENABLE to "y" and AIR_1WIRE_ENABLE to "y".
Notice:         None.
Relative doc:   None.
Example project:earbuds_ref_design.

FFT module usage guide

Brief:          This module is the Fast Fourier Transform (FFT) implementation for analyzing data.

Usage:          GCC:  For FFT, include the module with
                      1) Add the following module.mk for libs and source file:
                         include $(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/audio_fft/module.mk
                      2) Add the header file path:
                         CFLAGS  += -I$(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/audio_fft/inc

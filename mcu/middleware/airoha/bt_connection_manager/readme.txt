bt_connection_manager module usage guide

Brief:           This module is used to manage all the BR/EDR and aws connections.
Usage:           GCC: 
                      1) Add the following module.mk for include path and source file:
                         include $(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/bt_connection_manager/module.mk
                         in your GCC project Makefile.

Dependency:      None.
Notice:          None.
Relative doc:    None.
Example project: Please find the project earbuds_ref_design or headset_ref_design under the project folder.

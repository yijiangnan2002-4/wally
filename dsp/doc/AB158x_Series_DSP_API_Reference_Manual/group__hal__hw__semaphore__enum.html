enum hal_hw_semaphore_id_t

This enum defines the SEMAPHORE ID.


Note
In order to make sure the ID is dedicated for a specific core/feature, the user should use the predefined
MACRO that represents the SEMAPHORE ID. The MACRO is defined in the following lines:
mcu\driver\board\ab155x_evb\hw_resource_assignment\inc\hal_resource_assignment.h
dsp\driver\board\ab155x_evb\hw_resource_assignment\inc\hal_resource_assignment.h
Taking HW_SEMAPHORE_SYSLOG as an example, the user should define the same MACRO with the same ID in the
two header files above and use the MACRO when using the semaphore ID.
Enumerator
HAL_HW_SEMAPHORE_ID_0 

HW semaphore id 0.

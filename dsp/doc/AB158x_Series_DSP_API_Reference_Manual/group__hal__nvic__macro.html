Variable Documentation

int32_t nvic_irq_execution_number

This macro is used to query the current exception status of DSP.

If the value is less than zero, the DSP is running thread mode, otherwise an exception occured and the exception number is the value of the macro.

1 // sample 1: Judge in ISR handler mode.
2 if (HAL_NVIC_QUERY_EXCEPTION_NUMBER > HAL_NVIC_NOT_EXCEPTION ) {
3  // handler mode process
4 }
5 
6 // sample 2: Judge in thread mode.
7 if (HAL_NVIC_QUERY_EXCEPTION_NUMBER == HAL_NVIC_NOT_EXCEPTION ) {
8  // thread mode process
9 }
10 
11 // sample 3: Get the IRQn number.
12 int32_t irq_number;
13 
14 irq_number = HAL_NVIC_QUERY_EXCEPTION_NUMBER;

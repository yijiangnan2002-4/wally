enum preloader_pisplit_memory_type_t

This enum defines preloader malloc memory type, just for adaptability, user should configure which type is which memory.

Enumerator
PRELOADER_I_HIGH_PERFORMANCE 

Defines the type of high performance code memory for pisplit, like: DSP0 IRAM.

PRELOADER_D_HIGH_PERFORMANCE 

Defines the type of high performance data memory for pisplit, like: DSP0 DRAM.

PRELOADER_I_LOW_PERFORMANCE 

Defines the type of low performance code memory for pisplit, like: SYSRAM.

PRELOADER_D_LOW_PERFORMANCE 

Defines the type of low performance data memory for pisplit, like: SYSRAM.

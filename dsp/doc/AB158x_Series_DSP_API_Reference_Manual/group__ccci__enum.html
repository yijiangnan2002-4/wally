enum ccci_send_wait_type_t

This enum defines the CCCI send wait type.

Enumerator
CCCI_SEND_MSG_NO_WAIT 

Defines the wait type, means non-blocking to send.

The event will push to SW FIFO firstly, then automatic send the event when CCCI is idle

CCCI_SEND_MSG_WAIT_FOR_SEND_RECEIVE_DONE 

Defines the wait type, means will wait until other Core receive done and execute user callback done.

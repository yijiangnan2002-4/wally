PEQ lib module usage guide

Brief:          This module is the parametric equalization (PEQ) implementation for audio. It provides support for cutting or boosting a range of frequencies with filters.

Usage:          For the header files, please refer to the API referenece document in dsp\doc\.
                For the libraries, you do not directly call the libraries which are wrapped by dsp/$(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/dspalg/peq module. 
                For more information, please refer to dsp/$(SOURCE_DIR)/$(MIDDLEWARE_PROPRIETARY)/dspalg/peq/readme.txt
